# Dashboard Setup Guide

## Prerequisites

The Streamlit dashboard requires Python 3.8+ and several dependencies.

## Installation

### 1. Install Dependencies

```bash
pip install streamlit plotly pandas pillow
```

### 2. Run the Dashboard

```bash
streamlit run managertools/dashboard.py
```

The dashboard will start on `http://localhost:8501`

## Features

### Team Analysis View
- Team-level KPIs (size, code volume, commits, reviews, tickets)
- Individual member contributions table
- Productivity trends by sprint
- Team performance radar chart
- **Export individual performance reviews** (JSON, Markdown, PNG, PDF/HTML)

### Organization Overview
- Organization-wide metrics
- Team comparison table
- Top contributors (code volume, commits, reviews)

### Compare by Title/Role
- Cross-team comparison by job title (requires Backstage role data)
- Peer metrics for same-title professionals
- Productivity trends across titles
- **Export peer comparison reviews**

## Export Formats

From any view, select a team member and export their performance review:

- **JSON** - Structured data for programmatic use
- **Markdown** - Human-readable format for git repos/documentation
- **PNG** - Visual summary image (requires Pillow)
- **PDF/HTML** - Professional document (HTML can be converted to PDF with external tools)

## Data Sources

The dashboard reads from sprint report CSVs:
- `individual-{Team}-{User}.csv` - Individual contributor metrics
- Generated by `sprint_report_team_analysis.py` and `sprint_report_individual_analysis.py`

## Troubleshooting

**"No module named 'streamlit'"**
```bash
pip install streamlit
```

**"No module named 'plotly'"**
```bash
pip install plotly
```

**"No module named 'pandas'"**
```bash
pip install pandas
```

**"No module named 'pillow'"** (for PNG export)
```bash
pip install pillow
```

Or install all at once:
```bash
pip install streamlit plotly pandas pillow
```

## Notes

- Role/title data requires Backstage integration (see CLAUDE.md for configuration)
- Backstage must be configured in `~/.managerTools.cfg` with `backstageServer` key
- If Backstage is unavailable, "Compare by Title" view will show a message
- PNG export requires Pillow; other formats work without it
